Pegue este projeto Android Kotlin e conclua a integração real com o Spotify para uso pessoal no Android.

Tarefas:
- Adicionar a dependência correta da Spotify Android SDK / App Remote.
- Implementar autenticação/autorização adequada para mobile.
- Substituir o controller mock por implementação real.
- Fazer a conexão com o app oficial do Spotify.
- Implementar play por URI de playlist.
- Implementar pause, resume e leitura de status.
- Informar na UI quando o Spotify não estiver instalado.
- Melhorar a UI sem exagerar.
- Deixar instruções finais no README.

Requisitos de qualidade:
- Projeto deve continuar compilável.
- Código limpo, organizado e sem remover a estrutura atual sem necessidade.
- Explicar ao final quais arquivos foram criados/alterados e o que ainda depende das credenciais do usuário.
